Page-entity

Page - сущность для любой страници, как сообщества, так и пользователя

{
	_id: '4fg64dfg6d46d',
	type: 'comunity' or 'userpage',
	creator_id: 'sdd5fsdfsf6',
	moderator_ids: ['sdd5fsdfsf6', '4ff6sd56sfd6', '44sdf6sd4'],
}

- /home - главная страница
- /news - страница новостей

На страницах /home и /news выводятся посты из двух выбранных сообществ

! Новости на главной странице
	  формируются так:
	  - берутся 2 последние новости из сообщества 'ИКС' и выводятся в левый столбик
	  - берутся 2 последние новости из сообщества 'IT' и выводятся в правый столбик

! Новости на странице новостей
	  формируются так:
	  - берутся все новости из сообщества 'ИКС'
	  - берутся все новости из сообщества 'IT'
	  - сливаются
	  - сортируются по дате
	  - выводятся

Лайков, репостов и коментов пока не будет.
