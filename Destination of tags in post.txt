# Назначения тегов в посте

- ics - означает что это новость ИКСа
- it - означает что это новость в IT
! теги ics и it не должны быть в одном и том же посте, чтобы отделить новости ИКСа от новостей в IT;

- event -  обозначает будущее событие, мероприятие, конференцию;
- eventReport - обозначает отчёт о прошедшем событии, мероприятии, конференции; В отчёте желательно прикрепить ссылку на ВК-Альбом;
! теги ics и it не должны быть в одном и том же посте, чтобы

- tutorial - означает, что это обучающая статья по определённой технологии, например, по микросервисам в Java

- course - сообщение о том, что проводится курс от преподавателя из ИКСа;

в объекте post будет поля:
- eventData
- courseData

eventData: {
	date: "..."
}
courseData: {
	teachers: ["...", "..."]
}

У поста ещё будет поле type, обозначаещее тип поста
- type: 'comunity-post' - пост в сообществе
- type: 'user-post' - пост на странице пользователя
