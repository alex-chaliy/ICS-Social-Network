Назначения тегов в посте.txt

- event - событие, мероприятие, конференция;
- event_report - отчёт о событии, мероприятии, конференции; В отчёте желательно прикрепить ссылку на ВК-Альбом;
- course - курс от препода из ИКСа;

в объекте post будет поля:
- eventData
- courseData

eventData: {
	date: "..."
}
courseData: {
	teachers: ["...", "..."]
}
